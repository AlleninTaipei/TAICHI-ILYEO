# 太極一如

WT 跆拳道品勢圖解計畫

## 專案名稱由來

太極一場, 是太極品勢的第一套, 也是有色帶學員最早接觸的品勢; 一如, 是黑帶品勢的第九套, 對應九段, 象徵身心合一, 物我兩忘的最高境界. 從太極一場走到一如, 正好是跆拳道品勢系統的完整旅程, 因此以此命名本專案.

## 專案內容

- WT品勢介紹.md, 品勢系統的文字版參考資料, 包含太極品勢八套與黑帶品勢九套的名稱, 意涵與勵志語錄.
- WT品勢互動介紹.html, 將上述內容擴充為可互動瀏覽的網頁, 包含太極八卦輪盤與黑帶山水旅程等視覺化設計.
- Develop.md, 記錄本專案從零到完成的協作過程, 適合剛開始使用 Claude Code 的使用者參考.

## 如何使用

- 開啟 WT品勢介紹.md, 可直接閱讀純文字版的品勢資料表格.
- 用瀏覽器開啟 WT品勢互動介紹.html (直接雙擊檔案即可) , 瀏覽互動網頁版, 支援手機與電腦畫面.

## 內容範圍

- 太極品勢: 太極一場到太極八場, 每一場對應易經八卦中的一卦.
- 黑帶品勢: 高麗, 金剛, 太白, 平原, 十進, 地跆, 天拳, 漢水, 一如, 對應一段到九段.
- 補充說明: 太極品勢施行前, 曾經使用過的八卦品勢系統.

## 內容性質說明

- 品勢名稱, 卦象與哲學意涵, 整理自 WT 與國技院公開教材.
- 表格中的勵志語錄, 是根據品勢意涵延伸創作的原創內容, 並非官方文獻用語.
- 部分黑帶品勢的漢字寫法 (例如地跆, 天拳, 漢水) , 在不同中文資料來源中略有差異, 使用前建議與所屬道館或協會的教材核對.
